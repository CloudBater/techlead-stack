# Examples

Sample outputs from each skill/template, using **dummy data** (no real company information).

These show what each skill produces so you know what to expect before customizing:

| Example | Skill/Template | What It Shows |
|---------|---------------|---------------|
| `token-usage-report.md` | `/token-usage` | Weekly token/PR ratio, efficiency signals, 4-week trends |
| `pr-metrics-output.md` | `/pr-metrics` | 6-month PR analysis, per-person breakdown, slowest PRs |
| `1on1-prep-output.md` | `/1on1-prep` | Full meeting prep with activity chart, talking points, follow-ups |
| `retro-output.md` | `/retro` | Sprint scoreboard, per-person stats, action items, multi-sprint trends |
| `health-check-output.md` | `/health-check` | Dependency audit, lint/test status, CI builds, security scan |
| `shared-prompt-example.md` | `/team-knowledge-base` | What a shared prompt looks like (Django test generator, with iteration history) |
